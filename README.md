# hexlet-stats

> Check the user's statistics on Hexlet from the CLI.


## Install

```
$ npm install --global hexlet-stats
```


## Usage

```
$ hexlet-stats --help
  
  Usage
    $ hexlet-stats <login>
      
  Options
    --json -j     Output the result as JSON
 
  Examples
    $ hexlet-stats slavkluev
          
    Courses finished 24
        Independence 100.0%
              Points 1445
    Challenges count 10
              Rating #356
```
