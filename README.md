# hexlet-stats

> Check the user's statistics on Hexlet from the CLI.


## Install

```
$ npm install --global hexlet-stats
```


## Usage

```
$ hexlet-stats --help

  Usage
    $ hexlet <login>
   
  Options
    --json -j     Output the result as JSON
   	  
  Examples
    $ hexlet slavkluev
    
    Courses finished 24
        Independence 100.0%
              Points 1436
              Rating 253
```
